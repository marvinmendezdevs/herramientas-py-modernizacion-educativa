# Instrucciones de uso del script:

## Requermientos:

Para usar este script, debe tener instalado Python y a su vez, la librería Pandas y xlsxwriter.

pip install xlsxwriter
pip install pandas

Tener en la misma carpeta los archivos de estudiantes y docentes que da la app de los coreanos con sus nombre por defecto (student_cumulative..., teacher_cumulative...)

En la misma carpeta debe estar el archivo ce_bloques.csv <--- Este archivo servirá para filtrar centros escolares por bloques.

Ejecutar el script en main.py
